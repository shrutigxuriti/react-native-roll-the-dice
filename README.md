# react-native-roll-the-dice

React native component for roll the dice game

## Installation

```sh
npm install react-native-roll-the-dice
```

## Usage


```js
import { multiply } from 'react-native-roll-the-dice';

// ...

const result = await multiply(3, 7);
```
